Project Overview

Advanced computer vision system for real-time Baccarat card recognition

This project is a sophisticated real-time card recognition system designed specifically for Baccarat gameplay, utilizing state-of-the-art computer vision and machine learning technologies to automatically detect, classify, and track playing cards from live video feeds.

Built with C#/.NET 8 and WPF, the system integrates multiple YOLO architectures (v5, v7, v8) with advanced duplicate filtering algorithms and spatial card assignment logic, achieving 30 FPS real-time performance with 80% reduction in false positives.

Sophisticated multi-layer filtering system using IoU calculations, confidence-based selection, and configurable thresholds.
